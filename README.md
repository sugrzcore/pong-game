# Pong Game with AI (Normal Difficulty) - Python & Pygame

A simple 2D Pong game made with Python and Pygame.  
Play against a **normal AI opponent** with a **pink ball** and a scoring system. This project is suitable for graphics programming courses and small game development practice.

## Features
- Single-player mode against AI (normal difficulty)
- Pink ball
- Scoring system
- Smooth paddle and ball movement
- Works on Python 3 with Pygame

## Controls
- **Player Paddle (Left):**
  - W → Move up
  - S → Move down
- **AI Paddle (Right):** Automatic (Normal difficulty)

## How to Run
1. Make sure Python 3 is installed.
2. Install Pygame:
   ```bash
